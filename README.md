**愿景/version**   
记录以建筑作品为索引的线上社区如何改变着传统建筑设计的行为。

**范畴/scope**    
线上线下/O2O, 开源/open source, 云/cloud "互联网+"正在静悄悄的推进着由柯布西埃、格罗皮乌斯、密斯、赖特等[现代主义建筑](https://en.wikipedia.org/wiki/Modern_architecture)先锋创立的传统线下建筑设计行为。
